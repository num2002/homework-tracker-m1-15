# Homework Tracker ม.1/15

เว็บแอป Streamlit สำหรับติดตามการบ้านนักเรียน 40 คน โดยเก็บข้อมูลในไฟล์ `Homework_Tracker_M1_15.xlsx`

ระบบรองรับ 2 โหมด:

- **ใช้งานจริงออนไลน์:** Google Sheets + Streamlit Community Cloud
- **ทดสอบบนเครื่อง:** Excel (`Homework_Tracker_M1_15.xlsx`)

## ความสามารถ

- ติดตามนักเรียนด้วยเลขที่ 1–40 โดยไม่เก็บชื่อหรือรหัสนักเรียน
- เพิ่มการบ้านและสร้างรายการติดตามให้นักเรียนทุกคนอัตโนมัติ
- อัปเดตสถานะ: ยังไม่ส่ง, ส่งแล้ว, ส่งช้า, ยกเว้น
- Dashboard สรุปงานใกล้ครบกำหนด จำนวนส่งแล้ว/ค้างส่ง และเลขที่ค้างส่ง
- ผู้ปกครองเปิด Dashboard ได้ทันที ส่วนเมนูแก้ไขใช้รหัสผ่านครู
- สร้างข้อความภาษาไทยสำหรับคัดลอกส่ง LINE กลุ่ม
- สร้าง/ซ่อมโครงสร้างไฟล์ Excel อัตโนมัติเมื่อเปิดแอป

## ติดตั้งและรัน

ต้องมี Python 3.10 ขึ้นไป จากนั้นเปิด Terminal ในโฟลเดอร์นี้แล้วรัน:

```powershell
python -m venv .venv
.venv\Scripts\Activate.ps1
pip install -r requirements.txt
streamlit run app.py
```

เบราว์เซอร์จะเปิดที่ `http://localhost:8501`

## วิธีใช้งานทีละขั้น

### 1. นักเรียนในระบบ

ระบบเตรียมเลขที่ 1–40 ให้อัตโนมัติ ไม่ต้องกรอกชื่อหรือรหัสนักเรียน

### 2. เพิ่มการบ้าน

1. เปิดเมนู **เพิ่มการบ้าน**
2. ใช้ `HW_ID` ที่ระบบสร้างให้ หรือแก้เป็นรหัสของโรงเรียน
3. กรอกวิชา รายละเอียด วันที่สั่ง กำหนดส่ง และหมายเหตุ
4. กด **เพิ่มการบ้านและสร้างรายการ 40 คน**

ระบบจะเพิ่มสถานะ `ยังไม่ส่ง` ให้นักเรียนทั้ง 40 คนทันที

### 3. บันทึกการส่งงาน

1. เปิดเมนู **อัปเดตสถานะส่งงาน**
2. เลือกการบ้านจากรายการ
3. เปลี่ยนสถานะเป็น `ส่งแล้ว`, `ส่งช้า`, `ยกเว้น` หรือ `ยังไม่ส่ง`
4. ใส่วันที่ส่งและหมายเหตุได้ตามต้องการ
5. กด **บันทึกสถานะ**

หากเลือก `ส่งแล้ว` แต่วันที่ส่งเกินกำหนด ระบบจะเปลี่ยนเป็น `ส่งช้า` อัตโนมัติ

### 4. ดูภาพรวม

ผู้ปกครองเปิดเมนู **Dashboard** เพื่อดูจำนวนงาน งานที่ครบกำหนดภายใน 7 วัน จำนวนส่งแล้ว จำนวนค้างส่ง และเลขที่ค้างส่งแยกตามงานได้โดยไม่ต้องล็อกอิน

### 5. ส่งสรุปเข้า LINE

1. เปิดเมนู **สรุปข้อความส่ง LINE**
2. เลือกการบ้าน
3. ลากเลือกและคัดลอกข้อความในกล่องไปวางใน LINE หรือกดดาวน์โหลดไฟล์ข้อความ

### ตัวอย่างการทำงานประจำวัน

ตอนเช้าเพิ่มงานใหม่ → ระหว่างวันอัปเดตสถานะเมื่อรับสมุด → ก่อนเลิกเรียนเปิดหน้าสรุป LINE แล้วส่งรายชื่อค้างส่งเข้ากลุ่ม

## การใช้ไฟล์ Excel เดิม

วางไฟล์เดิมไว้ข้าง `app.py` และตั้งชื่อ `Homework_Tracker_M1_15.xlsx` แอปจะคงชีตเดิมไว้และเพิ่มชีต/คอลัมน์ที่จำเป็น หากไม่มีไฟล์ แอปจะสร้างใหม่พร้อมเลขที่ 1–40

ชีตหลักคือ:

- `Students` — เลขที่ 1–40 (คอลัมน์ชื่อและรหัสเดิมถูกเว้นว่างเพื่อรองรับไฟล์เก่า)
- `Homework` — ข้อมูลการบ้าน
- `Tracking` — สถานะรายคนต่อการบ้าน

## ข้อควรทราบ

- ปิดไฟล์ Excel ก่อนกดบันทึกในเว็บ เพื่อป้องกัน Windows ล็อกไฟล์
- ทุกครั้งที่บันทึก แอปจะเขียนไฟล์ชั่วคราวก่อนแล้วสลับไฟล์จริง เพื่อลดโอกาสไฟล์เสีย
- สำรองไฟล์ `.xlsx` เป็นระยะ โดยเฉพาะก่อนแก้ข้อมูลจำนวนมาก

## นำขึ้นออนไลน์ด้วย Google Sheets + Streamlit Cloud

ทำตามลำดับต่อไปนี้เพียงครั้งเดียว หลังจากนั้นเปิดแอปด้วย URL ได้จากทุกเครื่อง

### ขั้นที่ 1 — สร้าง Google Sheet

Google Sheet ของระบบถูกกำหนดไว้แล้ว:

```text
https://docs.google.com/spreadsheets/d/1JnxohQh-anm6HFJZudMOLHvnDl8osa-j2V7ynQhYQb0/edit
```

`GOOGLE_SHEET_ID` คือ `1JnxohQh-anm6HFJZudMOLHvnDl8osa-j2V7ynQhYQb0`

ไม่ต้องสร้างชีตย่อยเอง แอปจะสร้าง `Students`, `Homework` และ `Tracking` ให้อัตโนมัติ และจะนำเข้าข้อมูลจาก Excel ครั้งแรกให้ด้วย

### ขั้นที่ 2 — สร้าง Service Account ของ Google

1. เปิด [Google Cloud Console](https://console.cloud.google.com/)
2. สร้าง Project ใหม่ เช่น `homework-tracker-m115`
3. ไปที่ **APIs & Services > Library** แล้วเปิดใช้งาน:
   - Google Sheets API
   - Google Drive API
4. ไปที่ **IAM & Admin > Service Accounts**
5. กด **Create service account** ตั้งชื่อ `homework-tracker`
6. เปิด Service Account ที่สร้าง แล้วไปที่ **Keys > Add key > Create new key > JSON**
7. Google จะดาวน์โหลดไฟล์ JSON เก็บไฟล์นี้เป็นความลับ ห้ามส่งขึ้น GitHub
8. คัดลอกค่า `client_email` จาก JSON
9. กลับไป Google Sheet กด **Share** แล้วแชร์สิทธิ์ Editor ให้ `client_email` ดังกล่าว

### ขั้นที่ 3 — สร้าง GitHub repository

1. เข้าบัญชี GitHub `num2002`
2. สร้าง repository ใหม่ชื่อ `homework-tracker-m1-15`
3. ตั้งเป็น **Public** สำหรับ Streamlit Community Cloud แบบฟรี โดยเก็บเฉพาะโค้ด
4. ห้ามอัปโหลด `Homework_Tracker_M1_15.xlsx`, `.venv`, `node_modules`, ไฟล์ JSON ของ Service Account และ `.streamlit/secrets.toml`
5. ข้อมูลนักเรียนจริงต้องอยู่ใน Google Sheet ที่จำกัดสิทธิ์เท่านั้น

### ขั้นที่ 4 — Deploy บน Streamlit Community Cloud

1. เปิด [Streamlit Community Cloud](https://share.streamlit.io/)
2. เข้าระบบด้วย GitHub แล้วกด **Create app**
3. เลือก repository `num2002/homework-tracker-m1-15`
4. Main file path ใส่ `app.py`
5. เปิด **Advanced settings > Secrets**
6. เปิดไฟล์ `.streamlit/secrets.example.toml` แล้วแทนค่าตัวอย่างด้วยข้อมูลจริงจาก JSON
7. ตั้ง `APP_PASSWORD` เป็นรหัสผ่านสำหรับครู และตั้ง `GOOGLE_SHEET_ID` ผู้ปกครองไม่ต้องใช้รหัสผ่านเพื่อดู Dashboard
8. กด Deploy

เมื่อเปิดครั้งแรก แอปจะสร้างตารางใน Google Sheet และย้ายข้อมูลจาก Excel ให้โดยอัตโนมัติ

### ความปลอดภัย

- ห้าม commit ไฟล์ JSON ของ Service Account หรือ `.streamlit/secrets.toml`
- อย่าเปิด Google Sheet เป็น “Anyone with the link”
- แชร์ URL แอปให้ผู้ปกครอง ส่วนรหัสครูให้เฉพาะผู้ที่มีหน้าที่แก้ไขข้อมูล
- ใช้ปุ่ม **ดาวน์โหลดข้อมูลสำรอง Excel** ในแถบเมนูเป็นระยะ
